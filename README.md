Chuks Kitchen – Food Ordering App (UI/UX)

A mobile-first food ordering app designed to replace manual WhatsApp and phone-based ordering with a structured, user-friendly digital experience.

Project Goal

To create a fast, simple, and reliable ordering flow that allows users to:

Browse meals easily

Add items to cart

Choose Delivery or Pickup

Complete checkout confidently

The design prioritizes speed, clarity, and reduced friction.

 UX Approach

Browse without mandatory sign-in

Phone number + OTP authentication at checkout

Delivery & Pickup selection during checkout

Simplified cart and checkout flow

Clear feedback and confirmation states

Edge states included:

Empty cart

OTP error

Loading state

Order confirmation

 Key Assumptions

The app supports both Delivery and Pickup (not explicitly defined in BRD).

Authentication uses phone number + OTP for faster access.

Users can explore the menu before creating an account.

Basic loading and error handling are included to improve usability.
